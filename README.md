# Documentation for Inter Store - Inventory management system

Django Web application  for managing products and customer orders.

## Idea
Inter Store started as an Inventory management system and expanded to an online store. Main task is to have full functionality and options for an online store.

## Features

### Product management
- Product catalog with categories
- Product search form
- Product details page
- Product reviews
- Image upload for photos

### Order management
- Create and manage orders
- Automatic stock change and validation
- Cart functionality
- Change order status

### Review management
- Add product reviews
- All reviews page
- Average rating functionality

### Error Handling
- Custom 404 page
- User-friendly validation messages for all forms
